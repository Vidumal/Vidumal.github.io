Online Bookstore Management System

Project Overview

The Online Bookstore Management System is a web-based application built using Spring Boot and Java 17. The unique challenge of this project was to build a complete platform (registration, shopping cart, payments) without using a traditional SQL database. Instead, we utilized advanced file handling to persist data, demonstrating core OOP principles and MVC architecture.

Technologies Used

  • Java 17
  • Spring Boot
  • Apache Tomcat 9.0
  • JSP (JavaServer Pages)
  • File I/O Persistence
  • MVC Architecture

System Architecture (MVC)

Controllers

  • HomeController: Manages landing page & navigation.
  • UserController: Handles auth (login/register).
  • PaymentController: Processes transactions.
  • OfferController: Manages discounts.

File-Based Database

  • /data/*.txt: Stores Books, Users, and Offers as text files.
  • /data/image/: Stores product images locally.
  • Service Layer: Reads/Writes to these files instead of SQL queries.

Key Takeaway

This project highlights the ability to design robust logical architectures without relying on standard database management systems, showcasing deep understanding of Java I/O streams and object manipulation.
