HoneyGuard Pro

Advanced Threat Intelligence Honeypot

View Source Code

Project Overview

HoneyGuard Pro is a multi-threaded network honeypot designed to detect, log, and visualize unauthorized access attempts in real-time. What sets it apart is its unique Dual-Mode Architecture, which automatically adapts to its environmentâ€”offering a full GUI for local monitoring or a lightweight CLI for Docker/Cloud deployments.

Technologies Used

  • Python 3.9+
  • Docker
  • Tkinter (CustomTkinter)
  • Socket Programming
  • IP-API (Geolocation)
  • Threading

Key Features

GUI Interactive Dashboard

  • Live Attack Map: Visualizes attacker locations instantly on a zoomable world map using TkinterMapView.
  • Real-Time Stats: Dynamic counters for threat levels and total attacks.
  • Simulation Tool: Built-in tool to simulate SQLi and XSS attacks for testing.

CLI Headless Docker Mode

  • Auto-Detection: Automatically switches to CLI mode when running in a container.
  • Cross-Platform: Handles file paths for both Windows and Linux environments.
  • Instant Logging: Forces unbuffered output to Docker logs for immediate verification.

Threat Intelligence

  • Geolocation Tracking: Fetches Country, City, and GPS coordinates of attackers using ip-api.com.
  • Heuristic Analysis: Analyzes payloads to classify threats (SQLi, RCE, Recon) and assigns risk scores.
  • Forensic Logging: Automatically appends timestamped evidence to honeyguard_logs.txt for auditing.

Deployment

HoneyGuard Pro can be deployed locally on Windows for a visual monitoring experience or containerized via Docker for scalable cloud security monitoring.
