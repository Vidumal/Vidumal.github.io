SSH Honeypot

View Source

Project Overview

This project is a Low-Interaction Honeypot developed in Python using Socket programming. It is designed to mimic a vulnerable SSH server to attract malicious actors. Instead of blocking attacks, it silently logs the intruder's IP address, username, and password attempts, providing valuable Threat Intelligence on brute-force patterns.

Technologies Used

  • Python 3
  • Socket Programming
  • Paramiko (SSH Library)
  • Threading
  • Logging Module

How It Works

1. The Lure

The script binds to Port 2222 (or 22) and listens for incoming connections. To an external scanner (like Nmap), it appears to be a legitimate, active SSH service.

2. Data Capture

When an attacker attempts to login, the script accepts the connection but rejects the password. Meanwhile, it records the Source IP, Username, and Password to a local log file.

Key Features

  • Multi-Threaded: Handles multiple attacker connections simultaneously using Python Threading.
  • Stealth Mode: Does not reveal true system information to the attacker.
  • Forensics Log: Generates a structured `intrusions.log` file for analysis.

Educational Value

Building this honeypot provided deep insights into network protocols, socket handling, and the methodology of brute-force attacks in the wild.
