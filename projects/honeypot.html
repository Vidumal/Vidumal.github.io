CyberSteg Pro

View Source

Project Overview

CyberSteg Pro is a high-security steganography application designed to hide sensitive information within digital images. Developed as a specialized cybersecurity tool, it goes beyond basic data hiding by implementing Multi-Layered Security, ensuring that hidden messages are invisible to the eye and mathematically protected against extraction.

Security Stack

  • Python 3.10+
  • AES-256-GCM
  • PBKDF2 HMAC
  • CustomTkinter (GUI)
  • Pillow (Image Processing)
  • Coded Pathing (PRNG)

Multi-Layer Protection

1. The Vault

Data is first encrypted using AES-256-GCM. An authentication tag is generated to ensure 100% data integrity and prevent tampering.

2. The Maze

Instead of linear storage, the bits are scattered across the image using a Seeded PRNG algorithm. Without the key, the data path is impossible to reconstruct.

Advanced Features

  • Lossless Preservation: Disables PNG optimization to ensure zero bit-drifting during the save process.
  • Authenticated Decryption: Instantly detects if an image has been altered or a wrong key is used.
  • Stealth UI: Modern dark-mode interface with separate tabs for encryption and extraction workflows.

Technical Challenge

A major milestone in this project was overcoming pixel corruption. By forcing 3-channel RGB mapping and bypass-level list processing, the tool ensures bit-perfect reliability for the AES-GCM tag.
