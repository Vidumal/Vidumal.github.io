Face Mask Detection System

View Source

Project Overview

This project is a real-time computer vision application designed to detect whether a person is wearing a face mask or not. Utilizing Deep Learning and the MobileNetV2 architecture, the system classifies video frames into two categories: "With Mask" and "Without Mask" with high accuracy.

Technologies Used

  • Python
  • TensorFlow / Keras
  • MobileNetV2
  • OpenCV
  • NumPy
  • Kaggle Datasets

Workflow & Methodology

1. Data Preprocessing

  • Resizing: All images converted to 224x224 pixels.
  • Normalization: Pixel values scaled to [-1, 1] range.
  • Augmentation: Applied rotation, zoom, and horizontal flips to increase dataset variety.

2. Model Training

  • Base Model: MobileNetV2 (Pre-trained on ImageNet).
  • Head Model: Custom Fully Connected layers added for our specific 2-class problem.
  • Optimization: Used Adam optimizer with binary cross-entropy loss.

Key Features

  • Real-Time Detection: Process video streams instantly using OpenCV.
  • High Accuracy: Leverages Transfer Learning for robust performance even with limited data.
  • Lightweight: MobileNetV2 is optimized for speed, making it suitable for edge devices.

Impact

This system can be deployed in public spaces (entrances to malls, offices, airports) to automate compliance monitoring, ensuring public safety during health crises without manual intervention.
